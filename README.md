# Ram Suhrut's Personal-Website

Here is my personal website, which highlights my accomplishments, experiences, and journey.

Main goal was to create a website that integrates "flowy" elements and relies on immersive scroll-triggered animations, organic curves, and liquid morphing gradients.

# Elements

## Organic Navigation (The Flow)

- Physics-Based Inertia: Scrolling should mimic natural fluidity (like floating in zero gravity), where elements glide and bounce smoothly instead of stopping abruptly.
- Curved Grids: Reject rigid, boxy layouts. Use bezier curves, wave patterns, and circular blobs to divide sections.
- Path Following: Content reveals itself by following a continuous, winding line (or a "Spirograph" loop) rather than a straight vertical drop.

## Immersive Visuals & Color

- Liquid Gradients: Slow-moving, morphing, multi-color blurs create an ambient, bioluminescent aesthetic (e.g., iridescent purples, deep space blues, and neon teals).
- Glassmorphism: Use translucent, frosted-glass panels with soft blur effects and subtle neon underglows to layer information gracefully.
- Micro-Interactions: Elements react to the mouse cursor like magnetic fields, rippling or shifting shape when hovered over.

## Typography & 3D Elements

- Kinetic Type: Text that warps, stretches, or floats as the user scrolls.
Interactive 3D Objects: Rendered elements that the user can rotate or manipulate without leaving the page.
- Distortion Effects: Implement WebGL liquid distortions that make the entire webpage look like it is floating underwater or on an alien atmosphere.
